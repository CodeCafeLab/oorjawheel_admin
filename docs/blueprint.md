# **App Name**: Oorja Admin

## Core Features:

- Dashboard: Dashboard overview of key metrics (orders, customers, products, reports).
- Customer Management: Customer management section with filtering and pagination.
- Product Management: Product inventory management with add/edit forms.
- Order Management: Order tracking and management.
- Reports and Analytics: Data Visualization of the key data points with charts and graphs.
- Settings: Settings Panel for configuration and profile management.

## Style Guidelines:

- Primary color: Deep brown (#BE5A09) to reflect the brand's identity.
- Background color: Very dark desaturated brown (#0A0A0A) for a modern, professional look.
- Accent color: Light tan (#F2B777) to provide contrast and highlight key actions.
- Headline font: 'Forum', serif, to give the headers an elegant and strong touch. Note: currently only Google Fonts are supported.
- Body font: 'Questrial', sans-serif, for clean and readable text throughout the admin panel. Note: currently only Google Fonts are supported.
- Use a consistent set of line icons throughout the UI, with active states highlighted in the primary color.
- Responsive layout with a collapsible sidebar, fixed header, and flexible content area.
- Subtle hover effects on buttons and interactive elements.
